# 行为准则

Favewise 希望成为一个清晰、尊重、建设性的开源项目。

## 期望行为

- 保持尊重，反馈尽量具体。
- 默认善意，但也认真看待实际影响。
- 讨论问题本身，不攻击个人。
- 对不同背景、语言和经验水平的贡献者保持耐心。
- 安全问题请按 [SECURITY.zh-CN.md](./SECURITY.zh-CN.md) 报告，不要发到公开讨论区。

## 不接受的行为

- 骚扰、威胁或人身攻击。
- 针对身份、背景、能力或经验的贬损性语言。
- 未经许可公开他人私人信息。
- 反复发布离题、敌意或破坏性评论。
- 在项目空间发布垃圾信息、诈骗内容或无关推广。

## 执行

维护者可以编辑、隐藏或删除评论，关闭 Issue 或 Pull Request，屏蔽用户，或采取其他合理措施来维护项目空间。

如果你看到违反本准则的行为，非敏感问题可通过 GitHub Issues 联系维护者；敏感问题请使用 [SECURITY.zh-CN.md](./SECURITY.zh-CN.md) 中列出的安全/联系方式。
