# Code of Conduct

Favewise aims to be a constructive, respectful open source project.

## Expected Behavior

- Be respectful and specific.
- Assume good intent, but take impact seriously.
- Keep feedback focused on the work, not the person.
- Be patient with contributors from different backgrounds, languages, and levels of experience.
- Report security issues through [SECURITY.md](./SECURITY.md), not public threads.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Demeaning language related to identity, background, ability, or experience.
- Publishing private information without permission.
- Repeated off-topic, hostile, or disruptive comments.
- Using project spaces for spam, scams, or unrelated promotion.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests; block users; or take other reasonable action to keep project spaces healthy.

If you see behavior that violates this code, contact the maintainers through GitHub Issues if it is not sensitive, or through the security/contact channels listed in [SECURITY.md](./SECURITY.md) if it is sensitive.
